4  Modeling other kinds of data

This really only scratches the surface when it comes to modeling with data. Simple and multiple linear regression are useful tools, but only for predicting between continuous numeric variables that meet the criteria we discussed earlier in the section.

Generalized Linear Models

The Generalized Linear Model is the non-parametric version of the linear model, and can be accessed using the glm function, and allows you to specify a family of distributions other than normal in your assessment of the data.

For example, the “ropey” variable in the scat dataset is a binary variable, expressed as 0 or 1. If we wanted to perform a regression on this data, because it’s residuals are not distributed normally. For example, let’s say we want to know if there is a predictive relationship between mass and ropeyness.

library(modeldata)
ggplot(scat,aes(x=Length,y=ropey)) + 
  geom_point()

We can see that there appears to be some difference between ropey (1) and non-ropey (0) scat in terms of their length. However, it would be impossible to say that we could draw a line , Additionally, such a line would predict values for ropeyness that could

We can use glm to model this using a binomial logistic distribution:

scatGLM<-glm(ropey~Length,data=scat,family=binomial)
summary(scatGLM)

The coefficients and p-values here are similar to those from linear modeling. The Null Deviance measures of how well the response variable (here ropey) is predicted by the intercept only, while the Residual Deviance tells us how well it is predicted by the response/predictor variables (here just Length). Lower values of the latter indicate a better prediction from the response value.

The AIC (Akaike Information Criterion) is a metric that can be used for model comparison. It balances how well the model fits vs the number of parameters (variables) used. The lower the number, the more effective the model. For example, we might try the same analysis with a different predictor variable, like degree of taper:

scatGLM<-glm(ropey~TI,data=scat,family=binomial)
summary(scatGLM)

These models can be plotted using the method="glm" argument. To tell R what kind of , we add method.args to add an additional argument specific to glm. This takes a list, and the list will have one object called family, and it will store the character value “binomial”:

ggplot(scat,aes(TI,ropey)) +
  geom_point() +
  geom_smooth(method="glm", method.args = list(family = "binomial"),se=FALSE)

The glm function provides a number of different families of model, such as “Poisson” for count data with integers, or “Gamma” for continuous data with a positive skew.

Going further

If you’re interested in looking into doing more complex modeling, there are a number of resources out there. Here are a few to get you started:

  • There is an excellent section on regression in YaRrr! The Pirate’s Guide to R by Nathaniel Phillips, including a section on using the Generalized Linear Model.

  • For a treatment on non-linear regression techniques, the book Introductory Biostatistics with R has a couple of chapters focused on dealing with linear and non-linear models.

  • For machine learning classification techniques like random forests and gradient boosting, check out Hands On Machine Learning with R by Bradley Boehmke and Brandon Greenwell.

  • If you’re interested in doing more complex modeling and integrating modeling within the tidyverse workflow, I highly recommend Tidy Modeling with R by Max Kuhn and Julia Silge. The functions take on a very different structure, but once mastered can be very powerful for modeling data.
